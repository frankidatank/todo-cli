## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd todo-cli
  cd todo
```

Start virtual env

```bash
  python3 venv .venv
  source .venv/bin/activate
```

Install dependencies
```bash
  pip3 install -r requirments.txt
```

Start the app

```bash
  python3 todo.py 
```
